# 🚀 BuildCore

BuildCore is a **student-driven tech learning community** focused on:

- Coding  
- Robotics  
- Cybersecurity  
- Hardware experimentation  

Our goal is to help students **learn technology by building real projects**.

---

## 💫 About Me
🔭 I’m currently working on **BuildCore**, creating online classes and projects for students interested in tech.  
👯 I’m looking to collaborate on student tech communities, open-source learning projects, robotics builds, and educational tools.  
🤔 I’m looking for help with building scalable learning platforms, improving web development skills, and open-source collaboration.  
🌱 I’m currently learning Python, web development, cybersecurity fundamentals, and microcontroller-based hardware projects.  
💬 Ask me about coding basics, robotics projects, and learning tech as a student.  
⚡ Fun fact: I love experimenting with hardware and trying to run powerful tech on small devices.
